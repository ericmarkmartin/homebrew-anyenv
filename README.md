# homebrew-anyenv
A Homebrew Formula for [anyenv](https://github.com/riywo/anyenv).


# Usage
with [homebrew](http://brew.sh/), first tap this repository

```
$ brew tap matsub/anyenv
```

After that, install anyenv

```
$ brew install anyenv
```
